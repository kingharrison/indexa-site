How It Works

From documents to answers
in four steps.

01 ——
📦

Install Ollama

Download Ollama (free, open source) and pull an embedding and chat model. Indexa auto-connects on launch — no configuration needed.

02 ——
📁

Add Your Documents

Drag in PDFs, Markdown, DOCX, TXT, RTF, and HTML — or paste a URL to scrape a page or crawl an entire site with configurable depth.

03 ——
💬

Ask Anything

Ask natural language questions. Get AI-generated answers grounded in your documents, with source citations and relevance scores.

04 ——
📤

Export & Share

Bundle collections into portable .indexa files. Encrypt with AES-256. Share your knowledge without sharing your sources.

Features

Everything you need.
Nothing in the cloud.

📄

Multi-Format Ingestion

Supports PDF, TXT, Markdown, RTF, DOCX, and HTML. Drag and drop files directly into the app. AI-generated document summaries at ingest time.

Free
🌐

Web Scraping & Crawling

Paste a URL to index a single page, or crawl an entire site with configurable depth and page limits. Set auto-refresh schedules hourly, daily, or weekly.

Free
🗂

Collections

Organize documents into named collections, each with its own system prompt and query context. Duplicate and rename inline. Enable or disable documents individually.

Free
🔒

Collection Protection

Password-protect collections in two modes: Sources Masked (answers without citations) or Read Only (view and query, no modifications).

Free
✨

AI Distillation

Let AI rewrite documents optimized for retrieval accuracy. Toggle between original and distilled versions per document. Batch distill entire site crawls.

Free
🔌

HTTP REST API

Built-in local HTTP server exposes your RAG collections to any app. Bearer token auth, CORS restricted to localhost, rate limiting on password attempts.

Free
📦

Encrypted Bundle Export

Export collections as portable .indexa files with AES-256-GCM encryption. Recipients get the intelligence — never the source documents.

Pro
🤝

Multi-Provider Support

Works with Ollama by default or any OpenAI-compatible server like LM Studio. Configure multiple providers and switch chat models on the fly.

Free
💡

Source Citations

Every AI answer includes source citations with relevance scores, so you always know exactly which document each piece of information came from.

Free

Under the Hood

Built native.
No dependencies.

Platform
macOS · Native SwiftUI · Swift 6
Database
SQLite3 (raw C API, no ORM dependencies)
Vector Search
Cosine similarity, computed in-memory
Embeddings
nomic-embed-text via Ollama (768-dim vectors)
Encryption
AES-256-GCM (CryptoKit) + PBKDF2 (CommonCrypto)
HTTP Server
Apple Network framework (NWListener)
License Signing
P256 ECDSA asymmetric signing (CryptoKit)
Compression
LZFSE (Apple built-in)
